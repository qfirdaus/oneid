# Release OneID v2.1.1 — Controlled Pilot dan UI

**Versi:** 2.1.1
**Tarikh:** 16 Julai 2026

Release ini merekod kejayaan Controlled Pilot External Sync bagi subset tetap
2 New dan 1 Update tanpa tindakan destructive. Apply, pilot dan sync engine
telah dikembalikan kepada keadaan disabled selepas pelaksanaan.

Perubahan utama turut merangkumi backup/restore serta isolated rehearsal,
struktur deployment dan secret runtime yang berada di luar public root,
notifikasi toast top-right, confirmation SweetAlert, pagination Audit Log,
susunan date picker yang lebih padat dan pembuangan aset CSS legacy yang tidak
digunakan.

Paparan Version Releases kini menggunakan accordion eksklusif. Release terkini
terbuka secara default; release lama tertutup dan hanya satu release boleh
dibuka pada satu masa.

Tiada pengaktifan semula Apply dibuat oleh release metadata ini.
